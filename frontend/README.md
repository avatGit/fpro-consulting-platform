# Frontend - F-PRO Consulting Platform

This directory contains the React frontend for the F-PRO platform.

## Demo Accounts

If you have run the backend demo seeder (`npm run db:seed:demo` in the `Backend/` folder), you can use the following credentials to login at `http://localhost:3000/login`:

| Role       | Email                | Password      |
|------------|----------------------|---------------|
| **Admin**  | `admin@demo.fpro`    | `Admin@2026`  |
| **Agent**  | `agent@demo.fpro`    | `Agent@2026`  |
| **Client** | `client@demo.fpro`   | `Client@2026` |

For more comprehensive documentation, please refer to the main `README.md` at the root of the project.
