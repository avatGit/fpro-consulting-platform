## 🧪 Quick Test Guide

### 1. Dashboard Module
- View total users, orders, revenue, maintenance stats
- Check activity timeline

### 2. Users Module (Utilisateurs)
- See all users in a table
- Use search and filters (role, status)
- Select users and use bulk operations
- Edit or toggle user status

### 3. Quotes Module (Devis)
- View all quotes
- Filter by status
- Approve or reject pending quotes

### 4. Invoices Module (Factures)
- See invoice statistics
- View all invoices
- Mark invoices as paid
- Filter by status

### 5. Audit Logs (Logs d'audit)
- View all admin actions
- Filter by action type or resource
- See who did what and when

### 6. Settings (Paramètres)
- View system settings by category
- Update setting values
- Initialize default settings

### 7. Orders (Commandes)
- View all orders
- Filter by status
- Create invoices from delivered orders

### 8. Products (Produits)
- View all products
- Filter by type
- See stock levels

### 9. Maintenance
- View maintenance requests
- Filter by status
- See priority levels

## 🎨 What You'll See

- **Modern sidebar** with all 9 modules
- **Statistics cards** with gradients and animations
- **Data tables** with search, filters, and pagination
- **Action buttons** for quick operations
- **Status badges** with color coding
- **Responsive design** that works on all screens
